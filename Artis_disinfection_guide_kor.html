공정 개요 및 핵심 사항

소독 시퀀스 목적

각 소독 공정(린스·화학·열·CleanCart)은 유압 회로의 화학 잔류물 또는 세균을 제거하기 위해 설계된 단계별 시퀀스로 구성되며, 공정 종료 후 자동 린스가 실행됩니다.

알람 발생 시 동작

알람 발생 시 현재 단계가 중단되고 모든 펌프가 정지하며 2초 후 전자 밸브(EVDRAIN 제외)가 OFF됩니다. 알람 해제 시 해당 단계 처음부터 재시작됩니다.

유압 회로 장애

공기 유입, 튜브 꺾임, 내부 누수, 전도도 셀 드리프트 등 4가지 장애 유형이 기계의 예상치 못한 알람 또는 펌프 오류를 유발할 수 있습니다.

린스 공정 Rinse Process · Section 3.1.1
4개 항목
R-01
린스 공정 개요 Rinse Process Overview
유압 회로에서 화학 잔류물(소독제 또는 투석 농축액)을 제거하는 표준 린스 프로세스
최소 16분 물소비 9~50L 자동 실행
목적
유압 회로에서 화학 잔류물(소독제 또는 투석 농축액 잔류물)을 제거합니다. 공정 시간은 제거할 화학물질에 따라 달라지며 최소 1사이클로 투석 농축액 잔류물 제거에 충분합니다.
공정 시간 기준
기본 린스: 16분 (투석 농축액 잔류물 제거, 물소비 약 9L)
저과초산 화학 소독 후 린스: 27분 (물소비 약 17L)
과초산 화학 소독 후 린스: 38분 (물소비 약 25L)
차아염소산염 화학 소독 후 린스: 94분 (물소비 약 50L)
자동 시작 조건
화학/세정 프로그램 완료 후 자동 시작 / 기계 절전 또는 디스플레이 OFF 상태 해제 시 화학 소독제가 회로에 남아 있을 경우 자동 시작
참고: 유압 회로는 항상 울트라필터를 통과한 여과수로 세척됩니다. EVR1-EVS 라인은 예외로 수원에서 직접 물이 공급되어 Evaclean 라인을 통해 배수됩니다.

표준 린스 21단계
단계별 동작

Fill-up: 유압 회로 충전 (Fill-up phases 시퀀스)

Phase 7~8: 혼합 회로(EVPB, EVPA, PB, PSe, PA) 린스
Phase 9: EVD1 라인 린스
Phase 10~11: 외부 바이패스 및 2차 울트라필터 린스
Phase 12: 메인 플로우 패스 린스 (11~12단계 2회 반복)
Phase 13~15: 재순환 라인, 내부 바이패스, PS 펌프, EVW1, EVW2 린스
Phase 16: EVD2 라인 린스
Phase 17~20: 주 흐름 구성으로 2차 울트라필터, EVFLUSH/EVP, 혼합 회로 린스 (2~5사이클 시에만 실행)
Phase 21: EVC 또는 EVD 라인 린스
Draining + Phase 26~29: 배수 및 공정 종료

화학적 소독 공정 Chemical Disinfection Process · Section 3.1.2
4개 항목
C-01
화학적 소독 개요 Chemical Disinfection Overview
외부 소독제를 기계가 자동으로 희석하여 유압 회로를 소독하는 공정 (1:35 희석비)
소독제 1:35 희석 물소비 약 4.5L 35°C 고정
목적 및 원리
기계가 자동으로 소독제를 희석하여 설정 농도(1:35)를 달성합니다. 소독제는 기계 후면의 EVD 또는 EVC 전자 밸브를 통해 유입됩니다. 황색 커넥터: 차아염소산염 / 회색 커넥터: 과초산 소독제 사용.
공정 구성
예비 플러싱 및 희석: 약 11분
최소 접촉 시간(Dwell): 5분 / 최대 10분
소독 온도: 35°C (모든 소독제 공통)
물소비: 약 4.5L (린스 공정은 별도 계산)
참고: Phase 13과 14는 EVBHE 밸브도 함께 플러싱합니다.
C-02
화학적 소독 단계 시퀀스 Chemical Process Phase Sequence
Phase 0 → Fill_up → Phase 5~18 (소독제 분배) → Phase 19 (접촉 시간) → Phase 20 → 린스 순서로 실행
소독제 분배 20단계
단계별 동작
Phase 5: PS 펌프로 PS 루프 라인을 소독제로 충전, 메인 플로우 패스 고속 플러싱 (750~800ml/min)
Phase 6~7: 메인 플로우 패스 및 EVFLUSH 라인 소독제 분배
Phase 8~10: EVP 라인 소독
Phase 11: 혼합 회로(EVPB, EVPA, PB, PSel, PA) 및 EVD1 라인 소독
Phase 12~14: 외부 바이패스, 2차 울트라필터, 내부 바이패스 소독
Phase 15~17: EVW1, EVW2, EVD2 소독
Phase 18: PS 루프의 순수 소독제를 물로 희석 (IEC Plus 기기: 이 단계에서 P2 펌프 적응 계수 계산)
Phase 19: 접촉 시간 유지 (모든 액추에이터 OFF)
Phase 20: 모든 액추에이터 리셋 → 자동 린스 시작
C-03
소독제 종류 및 린스 구분 Disinfectant Types & Rinse Classification
과초산(38분 린스) / 저과초산(27분 린스) / 차아염소산염(94분 린스) 소독 후 린스 시간 자동 설정
3종 소독제 자동 린스 설정
과초산(Peracetic Acid)
최대 과초산 잔류량 0.10% → "과초산" 분류. 화학 소독 후 38분 린스 자동 설정.
저과초산(Low Peracetic Acid)
최대 과초산 잔류량 0.01% → "저과초산" 분류. 화학 소독 후 27분 린스 자동 설정. 정균(Bacteriostatic) 모드 지원.
차아염소산염(Hypochlorite)
화학 소독 후 94분 린스 자동 설정 (5사이클, 히터 활성화). 황색 커넥터 사용. 잘못된 소독제 감지 시 알람 578 발생 후 94분 린스 강제 실행.
빈 소독제 탱크 감지 알고리즘
Phase 19에서 전도도 셀(Cell_A, Cell_B, Diascan) 값을 읽고, 다음 린스 11단계에서 평균값을 비교합니다. 차이가 CCS 0.2mS/cm 미만 / IEC Plus 0.25mS/cm 미만인 경우 알람 533 발생 (탱크 비어있음).
C-04
화학 소독 알람 동작 Chemical Process Alarm Behavior
화학 소독 중 알람 발생 시 화학물질 낭비 방지를 위해 단계 재시작 시 남은 시간 기준으로 재개
알람 처리 화학물질 절약
린스와의 차이점
화학 소독 중 알람 발생 후 재시작 시, 단계의 남은 시간 기준으로 재개합니다 (린스는 처음부터 재시작). 화학물질을 불필요하게 소비하지 않기 위함입니다.
정균(Bacteriostatic) 모드
운영자가 정균 모드 선택 시 대기 시간 경과 후 기기 전원이 꺼지고, 다음 재시작 시 린스가 실행됩니다. 저과초산 또는 과초산만 사용 가능합니다.
열 소독 공정 Heat Disinfection Process · Section 3.1.3
4개 항목
H-01
열 소독 개요 Heat Disinfection Overview
열 효과로 기계를 소독. 온도는 HEAT A: 95°C / HEAT B: 99°C. 평균 소요 시간 약 34분, 물소비 약 11L
95°C / 99°C 약 34분 약 11L
통합 열 소독(Integrated Heat)
중앙 수처리 시설에서 90°C 이하(95°C 미만)의 물이 공급되는 경우 사용합니다. Water Plan 회로와 인렛 수관, 기계 전체를 소독합니다. EVFLUSH는 주기적으로 열려 공기 방울을 제거합니다.
초기 단계 목적
고유량(High Flow)으로 메인 회로를 플러싱하여 잠재적 침전물을 제거하고 소독 효과를 높입니다.
소독 순서
Phase 16에서 설정 온도 도달 → Phase 17: 1차 울트라필터 탄젠셜 소독 → Phase 18~29: 전체 유압 회로 소독 → Phase 30: 종료 전 전환 단계

열 소독 단계 시퀀스 Heat Process Phase Sequence
Fill_up → Phase 5~15 플러싱 → Phase 16 온도 도달 → Phase 17~30 소독 → 배수 / Phase 37 종료
온도 제어 30+단계
단계별 동작

참고: Phase 13과 30은 EVBHE 밸브도 함께 플러싱합니다.
H-03
온도 이탈 및 알람 처리 Temperature Deviation & Alarm Handling
Phase 18~28 중 온도 89°C 미만 하락 시 Phase 38(온도 회복 단계)로 이동하여 95°C 복귀 후 재개
온도 모니터링 89°C 임계값
온도 이탈 처리
Phase 18~28 실행 중 온도 89°C 미만 → Phase 38(온도 회복)으로 이동
Phase 38에서 95°C 도달 시 이전 단계로 복귀하여 재개
전원 오류 처리
TP > 90°C: Phase 39 실행 → 모든 펌프 정지, 2초 후 밸브 OFF → 5분 후에도 전원 미복구 시 자동 전원 OFF
TP < 90°C: Phase 39 건너뜀
사용자 중단 처리
사용자가 프로세스를 중단하면 배수 단계 실행 후 중단됩니다. 수동으로 시작한 열 소독은 공정 종료 후 배수를 건너뜁니다 (치료 사이에 시간 절약).
H-04
Autostart 기능 주의사항 Autostart Feature Notes
Autostart로 예약된 열 소독/통합 열 소독은 Heat with CleanCart 후 빈 CleanCart를 삽입한 채로 실행 가능
운영 주의 Autostart
Autostart 조건
Heat with CleanCart 성공적 완료 후 빈 CleanCart를 BiCart Holder에 삽입한 상태에서 Autostart로 예약된 열/통합 열 소독 실행 가능합니다.
주의 사항
Heat with CleanCart와 예약된 열 소독 사이에 사용자가 디스플레이 OFF 또는 저전력 상태에서 기기와 상호작용하면, 기기는 BiCart 암이 닫힌 상태에서만 공정을 실행합니다.

CleanCart 공정 개요 CleanCart Process Overview
구연산 또는 탄산나트륨이 든 CleanCart를 사용하여 열 효과로 소독 (95°C / 99°C). 평균 약 44분 소요

Phase 15: 90°C 도달 목표 (카트리지 프라이밍 시작 온도)
Phase 16: 온수로 카트리지 충전 및 유압 회로 잔류 전도도 세정
Phase 17A: CleanCart 용액을 메인 유압 회로로 분배 시작 (B 셀 전도도 확인)
Phase 18~26: CleanCart 용액 전체 유압 회로 분배 (EVD1, 2차 UF, EVW1, EVW2, 내부/외부 바이패스, EVD2, 혼합 회로)
Phase 27: 99°C 또는 95°C 도달 후 210초간 재순환 유지 → 1차 울트라필터 열 소독
Phase 28~40: 전체 유압 회로 소독 (EVFLUSH, EVP, EVD1, PS 루프, EVW1/2, 메인, 내부 바이패스, 2차 UF, 혼합, 외부 바이패스, EVD2)
Phase 41~43: EVP 라인, EVD1 라인, EVFLUSH 라인 플러싱 및 탈기 루프 냉각
참고: Phase 12, 21, 41은 EVBHE 밸브도 함께 플러싱합니다.

유압 회로 장애 유형 Hydraulic Circuit Failure Types · Section 3.2
4개 항목
F-01
유압 회로 내 공기 유입 Air Entering the Hydraulic Circuit
수원에서 공기 유입 또는 SelectBag/BiCart 연결부 공기 유입 시 BTP 상시 공기, PC 펌프 속도 저하, LFP 플로우 스위치 오동작 유발
공기 유입 PC 펌프 오류 전도도 오류
수원 공기 유입 경로
수처리 시설에서 물과 함께 공기가 방출될 경우, 공기 방울은 1차 울트라필터(습식 막으로 공기 통과 불가)에 의해 포집됩니다. EVFLUSH는 주기적으로 열려 "오염측"을 탈기하지만, 공기 방울이 많은 경우 울트라필터 여과 능력이 저하됩니다.
나타나는 장애 증상
BTP(버블 트랩) 읽기 항상 공기 상태
PC 펌프 속도 평균 대비 저하
LFP 플로우 스위치 상위 위치 유지 실패 → 가열·탈기 서브시스템 저유량 표시
P1 펌프 설정 유량 유지 불가
SelectBag/BiCart 공기 유입
SelectBag, SelectCart, BiCart 홀더 및 산 커넥터에서 공기가 유입되면, 혼합 챔버, 전도도 셀, P1 펌프를 통과합니다. BT1 버블 트랩이 모든 공기를 배출하지 못하면 D1c 유량계에도 공기 방울이 통과합니다. 린스/소독 중 EVPRIM 밸브만 통과하는 단계에서 공기가 혼합 회로로 유입되어 유량 제한, P1/P2 펌프 오류, 투석 압력 범위 이탈, 유량 설정 오류 등을 유발합니다.
F-02
유압 튜브 꺾임 (Kinking) Kinking of Hydraulic Tubes
실리콘/산토프렌 튜브가 꺾이면 부분적 또는 완전한 막힘 발생. BiCart 중탄산 라인 꺾임 시 전도도 알람 발생 가능
튜브 막힘 전도도 알람 P1 펌프 오류
BiCart 중탄산 라인 꺾임
준비 중 BiCart 카트리지 프라이밍 시 기계가 EVPRIM 및 EVPRIM1 밸브를 닫아 혼합 회로에 음압을 형성하고 공기를 배출한 후 EVPRIM1 밸브를 열어 BiCart 카트리지로 물이 유입됩니다. 이때 부분적으로 꺾인 중탄산 라인은 음압에 의해 완전히 접혀 막힘이 완성되고, 카트리지 내 수위가 예상보다 낮아져 전도도 알람을 유발합니다.
CleanCart 소독 시 영향
CleanCart 소독에서도 동일한 문제가 발생합니다. 유량 저하로 CleanCart 분말이 올바르게 용해되지 않아 전도도 측정 시 낮은 값이 감지되고 알람 504 ("CleanCart 공정 소독제 수준 낮음")이 발생합니다.
린스/소독 중 영향
EVPRIM 밸브만 통과하는 단계에서 꺾인 튜브가 유량을 제한합니다. P1 펌프가 유량 유지를 위해 속도를 높이면 P2 펌프도 속도를 높여 PFS, PO에 음압이 발생하고, 이는 P1 펌프 오류, 투석 압력 범위 이탈, 유량 설정 오류로 해석될 수 있습니다.
F-03
부품 내부 누수 Internal Leakage of Components
전자 밸브, 제한기, 펌프의 내부 누수. EVDEG 밸브 누수 시 PC 펌프 주파수 증가 및 PDG 압력 도달 실패로 알람 발생
밸브 누수 PC 펌프 오류
내부 누수 특성
내부 누수는 전자 밸브와 관련이 많으며, 제한기나 펌프까지 확장될 수 있습니다. 누수 정도는 명백할 수도 있고 매우 미세할 수도 있어 다양한 장애 조건을 만들어냅니다.
EVDEG 밸브 누수 예시
EVDEG 밸브 기능: PC 펌프가 PDG 센서 폐쇄 루프에서 작동 시 ORDEG 제한기를 우회하여 가열·탈기 서브시스템에 음압 형성
EVDEG가 설계상 열린 경우: PDG 압력 무관, PC 펌프는 고정 주파수로 구동 (주로 열 소독 중)
EVDEG 내부 누수 발생: PDG가 원하는 압력(보통 100mmHg 절대)에 도달하지 못함
결과: PC 펌프 주파수 증가 → PC 펌프 관련 알람 발생
F-04
전도도 셀 드리프트 Drift of Conductivity Cells
ΓcA 전도도 셀 드리프트 시 PA 세라믹 펌프 속도 오류 및 전도도 T1 테스트 실패 또는 전도도 알람 발생
셀 드리프트 PA 펌프 오류
드리프트 영향
PA 세라믹 펌프는 산 농축액을 끌어당기며 ΓcA 전도도 셀을 제어 루프로 사용합니다. ΓcA 셀의 드리프트는 PA 펌프 속도를 잘못 조절하여 목표 소듐 값에 도달하지 못하게 하고, 전도도 T1 테스트 실패 또는 전도도 알람을 유발할 수 있습니다.
불량 셀 확인 방법
방법 1 (비교법): 세 개의 전도도 셀(ΓcA, Γp, ΓcD)을 비교하여 다른 두 셀과 정렬되지 않은 셀을 확인
방법 2 (외부 참조): 전도도 테스트를 수행하고 전도도 셀 읽기를 외부 참조값과 비교
