Analysis Helper

>>> 파일 업로드 방법
드래그 앤 드롭 / 파일 선택

파일을 여기에 드래그하거나 클릭하여 업로드

지원 형식: .dat (Artis BBOX 블랙박스 파일)

여러 파일 동시 선택 가능

파싱 중... (65%)
방법 A — 드래그 앤 드롭
1
파일 탐색기에서 .dat 파일을 선택합니다
2
점선 박스 위로 파일을 끌어다 놓습니다 (박스가 보라색으로 변합니다)
3
마우스를 놓으면 자동으로 파싱이 시작됩니다
방법 B — 클릭하여 선택
업로드 영역을 클릭하면 파일 선택 대화상자가 열립니다. Ctrl+클릭 또는 Shift+클릭으로 여러 파일을 한 번에 선택할 수 있습니다.
>>> 주요 기능 상세
Alarm Log 탭
텍스트 검색 — 알람 설명(Description) 또는 파일명으로 실시간 검색
파일 필터 — 특정 .dat 파일의 레코드만 보기
유형 필터 — ALARM / OPERATOR_ACT / SET_CNF_VALUE / AUTOMATIC / DIAGNOSTIC 중 선택
컬럼 정렬 — DATE/TIME, TYPE, CODE 등 헤더 클릭으로 오름차순/내림차순 정렬
행별 CSV — 각 레코드 우측 ⬇ CSV 버튼으로 해당 행의 전체 파라미터 다운로드
Files 탭
파일 목록 — 업로드된 모든 .dat 파일 목록 표시
버전 뱃지 — 자동 감지된 펌웨어 버전 (8.09 / 8.33 / 8.60 / 9.05)
메타 정보 — 레코드 수, 시작 타임스탬프, 알람 카운트 표시
파일별 CSV — 해당 파일의 모든 알람 레코드를 하나의 CSV로 내보내기
오류 표시 — 파일 형식이 맞지 않을 경우 오류 메시지 표시
CSV 내보내기
Export All CSV — 현재 필터 조건의 모든 레코드를 하나의 CSV 파일로 다운로드
행별 CSV — 특정 타임스탬프의 전체 HYD/BLO/PRO 파라미터를 CSV로 다운로드
파일별 CSV — Files 탭에서 개별 파일의 모든 알람 레코드를 CSV로 내보내기
CSV 구조 — DATE/TIME, TYPE, CODE, DESCRIPTION + 해당 시점 전체 파라미터 컬럼 포함
Reset 버튼 — 모든 로드된 파일과 결과를 초기화하고 새 분석 시작

>>> 사용 팁 & 주의사항
💡 로컬 처리
모든 파일 처리는 브라우저 내에서만 이루어집니다. 파일이 서버로 전송되지 않아 환자 데이터 보안이 보장됩니다.
📂 다중 파일
여러 .dat 파일을 한 번에 업로드하면 통합 분석이 가능합니다. Files 탭 필터로 개별 파일을 선택해 확인할 수 있습니다.
🔄 초기화
다른 장비의 파일을 분석하려면 Reset 버튼을 눌러 초기화한 후 새 파일을 업로드하세요.
⚠️ 파일 형식
지원되지 않는 파일 크기의 .dat 파일은 오류 메시지와 함께 표시됩니다. 파일이 손상되지 않았는지 확인하세요.
📊 CSV 활용
행별 CSV에는 해당 시점의 모든 HYD/BLO/PRO 파라미터가 포함되어 심층 분석에 활용할 수 있습니다.
🌐 브라우저 권장
최신 버전의 Chrome, Edge, Firefox에서 최적으로 동작합니다. Internet Explorer는 지원되지 않습니다.
